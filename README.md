# weather-dashboard

## Functionality
Users can input the name of a city in the search bar and upon searching the OpenWeather API will be queried to get information on the current and future weather conditions of the city which will be displayed on the screen for the user.

Users can view their search history which will display previous searches which have been saved to local storage. Upon clicking a previous search that search term will be searched again displaying the weather for that city.
